# 📈 First ML: Predictive Analytics Web Application

A lightweight, interactive machine learning web application built using **Python**, **Scikit-Learn**, and **Streamlit**. The core predictive engine utilizes a supervised **Logistic Regression** classification model trained to evaluate input features and deliver fast, data-driven probability assessments directly through an intuitive UI.

🌐 **Live Production Link:** [https://firstml-7cpurhm9churdbxnlr3x6d.streamlit.app/](https://firstml-7cpurhm9churdbxnlr3x6d.streamlit.app/)

---

## 🚀 Key Features

* **Interactive Prediction Engine:** Users can input custom feature values using native Streamlit sliders and input forms to generate inferences on-the-fly.
* **Supervised Machine Learning:** Leverages a calibrated **Logistic Regression** classifier for clear binary/multi-class performance metrics.
* **Instant Cloud Inference:** Integrated directly with Streamlit Community Cloud for fast, continuous delivery directly out of the GitHub main branch.

---

## 🛠️ Tech Stack & Architecture

* **Language:** Python 3.10+
* **Machine Learning Library:** Scikit-Learn (Sklearn)
* **Data Processing:** Pandas & NumPy
* **Frontend Web Framework:** Streamlit
* **Deployment Cloud Environment:** Streamlit Community Cloud
